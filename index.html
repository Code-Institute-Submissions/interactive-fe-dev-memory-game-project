Game rules:

The game will start with one color lightening up

  1. Watch the computer lighten up the color
  2. Click on the same color

Each time you press the right color the computer will add one color to the series

In case you press the wrong color you lose one life and the computer replays the pattern which you missed

Good luck and enjoy!!!
